I'm trying to develop a better way to blog about code. I don't want to leave my natural environment (ie. command line, Xcode, TextMate, git, GitHub) and I want the posts to look really freaking good. My code posts on tumblr suck. And they're a pain in the ass to create.

I have an idea about using pure git and GitHub Pages to accomplish this. Just wrote up my plan in the README for this repository. Still in the discovery phase of this journey, but it seems workable. If you're so inclined, check out the readme for this repo and follow along as I fuck things up.

If eventually I get it right, I'll write an actual jekyl blog post about what I'm doing here and how I'm doing it. Until then, keep hacking.
